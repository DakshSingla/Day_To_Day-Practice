<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 1234<strong>

<pre><span style="font-size: 18px;"><strong>Input: </strong>n = 1243<strong>
Output: </strong>false<strong>
Explanation: </strong>Since 4 &gt; 3, the digits are not in non-decreasing order.</span></pre>
